EPFL course EE613 - Machine Learning for Engineers - Fall 2019

This course is given in collaboration with Jean-Marc Odobez and Francois Fleuret. You can find below my slides in pdf format.

The exercises for the labs are available here.

The program and material for the complete course is available here.


LINEAR REGRESSION I

Content:
  • Multivariate ordinary least squares
  • Singular value decomposition (SVD)
  • Kernels in least squares (nullspace)
  • Ridge regression (Tikhonov regularization)
  • Weighted least squares
  • Iteratively reweighted least squares (IRLS)
  • Recursive least squares
Material:
  • Lecture slides: EE613-linearRegressionI.pdf

LINEAR REGRESSION II

Content:
  • Logistic regression
  • Tensor-variate regression
  • Tensor methods
  • Hadamard, Kronecker, Khatri-Rao products
  • Canonical polyadic (CP) decomposition
  • Tucker decomposition
Material:
  • Lecture slides: EE613-linearRegressionII.pdf

HIDDEN MARKOV MODELS

Content:
  • Markov models
  • Hidden Markov model (HMM)
  • Covariance structures in mixture models
  • Forward-backward algorithm
  • Viterbi decoding (dynamic programming)
  • Expectation-maximization (EM)
  • Hidden semi-Markov model (HSMM)
  • HMM with dynamic features (Trajectory-HMM)
Material:
  • Preamble slides: EE613-HMM-preliminaries.pdf
  • Lecture slides: EE613-HMM.pdf
